

## Features
- Storage Accounts
  - Explore Blob Containers, File Shares, Queues and Tables
  - Copy Connection String
  - Copy Primary Key

- Blob Containers
  - Deeplink into Storage Explorer
  - Enumerate Blobs
  - Edit Blobs

- File Shares
  - Deeplink into Storage Explorer
  - Enumerate Files
  - Edit Files

- Queues
  - Deeplink into Storage Explorer

- Tables
  - Deeplink into Storage Explorer
